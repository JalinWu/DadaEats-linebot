# DadaEats linebot

### Usage

```sh
$ npm install
```

```sh
$ mkdir config

$ cd config

$ copy nul config.js
```

```js
// config.js
var channelData = {
    channelId: '',
    channelSecret: '',
    channelAccessToken: ''
};

module.exports = {
    channelData
};
```

```sh
$ npm run dev
```

```sh
$ ngrok http 3000
```
